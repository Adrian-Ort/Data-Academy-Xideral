Proyecto Final

Completado — Marzo 2026

NYC Yellow Taxi — Data Pipeline

Pipeline de datos end-to-end que descarga, procesa y visualiza los registros de viajes de taxi amarillo de Nueva York (NYC TLC), desplegado completamente en AWS. El sistema opera de forma totalmente automatizada: desde la ingesta de archivos parquet hasta un dashboard interactivo con Streamlit, sin intervención manual.

Dashboard
Arquitectura AWS
Fuente de datos NYC TLC — 59 meses procesados (~3M viajes/mes)
Procesamiento Apache PySpark 3.5 · limpieza + 5 tablas analíticas
Automatización Lambda (trigger S3 PUT) · watcher.py en EC2 (crontab 1 min)
CI/CD GitHub Actions runner en EC2 · docker-compose en push a main
AWS S3 AWS Lambda EC2 PySpark Streamlit Docker GitHub Actions Python 3.12
